# Dataset Notes

## 선택 기준

- 이 프로젝트는 실시간 흐름 feature를 직접 만들 계획이므로, 원시 패킷 데이터가 필요하다.
- 따라서 가능하면 `PCAP` 형식 데이터셋을 우선 사용한다.

## 현재 확인 사항

- CICDDoS2019 공식 안내는 raw `PCAP`과 flow `CSV`를 모두 제공한다고 설명한다.
- 공식 설명상 새 feature extractor를 사용할 경우 `PCAP` 사용이 적절하다.
- 현재 공식 다운로드 폼은 접속 시 서버 에러가 표시되어, 즉시 자동 다운로드 가능한 상태인지 추가 확인이 필요하다.

## 다음 행동

1. 공식 다운로드가 복구되는지 확인
2. 필요하면 대체 PCAP 공개 소스 검토
3. 확보한 PCAP 중 일부로 feature 추출 파이프라인부터 구현
